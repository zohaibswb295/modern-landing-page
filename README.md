# 🚀 TechNova — Digital Agency Landing Page

A modern, clean, and responsive landing page designed for a digital agency named **TechNova**.  
This project focuses on delivering a professional UI/UX experience using pure HTML and CSS.

---

## 📖 About The Project

TechNova is a digital agency concept website that highlights services, company introduction, and user engagement through a structured and visually appealing layout.

This landing page is designed with:
- Clean typography
- Soft color palette
- Structured sections
- Smooth user experience

## ✨ Key Features

✔ Modern and elegant UI design
